🌐 Basic Network Port Scanner

Python Cybersecurity Networking

This is a simple, fast, and efficient Network Port Scanner built using Python's built-in socket library. It helps identify open network ports on a target IP address.

How it works:

  • Attempts TCP connections to a specified range of ports.
  • Uses a short timeout for rapid scanning.
  • Handles exceptions like keyboard interrupts and unresolved hostnames gracefully.
